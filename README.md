# 📊 Sales-and-Revenue-Analysis-Dashboard-PowerBI

Interactive Sales & Revenue Analysis Dashboard built using PowerBI. Includes KPI tracking, sales trends, and product performance insights with slicers  for dynamic filtering.

## Project Overview
This project is an interactive Sales & Revenue Dashboard built using PowerBI.
It helps analyze sales performance, identify trends, and gain business insights.

## Objectives
- Analyze total sales and revenue
- Identify top-performing products
- Visualize sales trends over time
- Compare regional performance

## Features
- KPI Card for Total Sales
- Sales Trend Line Chart
- Top Products Analysis (Column Chart)
- Region-wise Sale Distribution (Pie Chart)
- Interactive Filters (slicers):
 - Region
 - Product
 - Year

## Tools Used
- Power BI
- CSV Datset

## Key Insights
- Identified top-selling products
- Observed sales trends across years
- Compared sales distribution by region

## Files Included
- '.pbix' file (power BI dashboard)
- Dataset (CSV)
- Dashboard screenshot (.png file)

## How to Use
1. Download the '.pbix' file
2. Open in Power BI Desktop
3. Use slicers to explore data interactively.
